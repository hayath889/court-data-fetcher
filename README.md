## Court Data Fetcher & Mini-Dashboard

### Court Targeted
[Ballari District Court](https://districts.ecourts.gov.in/ballari)

### Setup

1. Clone this repo
2. Create virtualenv
3. Run `pip install -r requirements.txt`
4. Run `python app.py`
